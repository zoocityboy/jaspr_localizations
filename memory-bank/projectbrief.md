# Project Brief: Jaspr Localizations

`jaspr_localizations` is a Dart library designed to provide internationalization (i18n) and localization (l10n) support for applications built with the **Jaspr** web framework.

## Core Features
- Type-safe string generation from `.arb` files.
- Integration with `jaspr` framework components.
- Support for dynamic locale switching.
- Browser language detection.
